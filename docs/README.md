# GitHub Pages frontend

Este diretório contém a versão estática da interface web.

Uso esperado:

1. publique este repositório no GitHub
2. ative GitHub Pages com source em `GitHub Actions`
3. faça push para `main` ou `master`
4. acesse `https://SEU_USUARIO.github.io/NOME_DO_REPO/`

O frontend chama a API na VPS via `fetch` e permite trocar o endpoint no campo `API Base URL`.
